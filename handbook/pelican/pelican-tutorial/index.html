Pelican Tutorial

Tutorial for Pelican, static site generator, written in Python.

python3 install from pip3

pip3 install pelican markdown

Quick initial project

pelican-quickstart

Setup pelicanconf.py for local developing

RELATIVE_URLS =True

Run develop server

make devserver
pelican --list

Build with options

pelican --listpelican -s pelicanconf.py -o output -D

Download templates

git clone --recursive https://github.com/getpelican/pelicanthemes.git

Select theme, copy to /theme/ on root project and set you THEME on pelicanconf.py for customisation

THEME='/theme/{{your_theme}}/';
