Change file encoding

Сheck and change file's encoding from the command-line in Linux with iconv.

Check.

file -i input.file

Change encoding, example from ISO-8859-1 to UTF-8

iconv -f ISO-8859-1 -t UTF-8 input.file -o out.file

Check encoding out file.

file -i out.file
