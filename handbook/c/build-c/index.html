Build simple programm on C

How-to build simple programm on C

gcc -o hello_world hello_world.c

Compile gtk3

gcc `pkg-config --cflags gtk+-3.0` -o program program.c `pkg-config --libs gtk+-3.0`

Compile OpenGl

gcc program.c -o program -L/usr/X11R6/lib/ -lGL -lGLU -lglut

Check memory leak

valgrind --leak-check=full ./program
