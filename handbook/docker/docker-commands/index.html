Docker Commands

Most used.

Build image

docker build . -t tagname

Shell inside container, etc...

docker exec -it image sh

Show logs

docker logs image 2>&1 |  grep "string"

Get dump.sql from container

docker exec -it image pg_dump -U username database &gt; dump.sql

Restore from dump.sql to container

cat dump.sql | docker exec -i image psql -U username -d database

Clear old image

docker system prune -a

Reload image and run demon in docker-compose

docker-compose stop image
docker-compose rm image
docker-compose up -d image
